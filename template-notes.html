O melhor gerenciador de proxy reverso

Essencial para rodar os templates do One Click Stacks esse é o que vai controlar o backend da aplicação e que vai apontar todas as entradas dos subdominios para os containers.

Documentação Traefik.

Guidelines para deploy Traefik

Via linha de comando bash (Recomendado)

Defina as variaveis subdominio e dominio e email da ACME antes de instalar a stack.

ACME_EMAIL=seuemail@gmail.com \\
docker stack deploy -c local_da_stack/traefik.yml traefik

Via clonagem de template no Portainer

Volte na tela anterior e clique em Copy as Custom.
Salve o template como seu colocando um titulo e descrição.

Depois clique em Stacks e de um nome para sua Stack.
Logo abaixo clique na quarta opção em Custom Template e selecione o template salvo.
Clique no botão add an enviroment variable e adicione a variável abaixo.

Name: ACME_EMAIL
Value: seuemail@gmail.com

Após inserir as variáveis no seu sistema é so clicar em Deploy the Stack.

Guidelines para deploy n8n

Defina as variaveis subdominio e dominio e email da ACME antes de instalar a stack.

SUBDOMAIN_N8N=n8n
DOMAIN=seudominio.com.br
ACME_EMAIL=seuemail@gmail.com

Defina as variaveis de SMTP se quiser adicionar mais usuarios no futuro, tambem pode ser feita em Services >> n8n >> seção enviroment com o container rodando.

SMTP_SERVER=smtp.gmail.com
SMTP_PORT=465
SMTP_USER=email@dominio.com.br
SMTP_PASSWORD=suasenhasupersegura SMTP_SENDER=naoresponda@seudominio.com

Após inserir as variáveis no seu sistema é so clicar em Deploy the Stack
